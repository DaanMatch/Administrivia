ArcGIS Online¬∂

What is GIS?

A Geographic Information System (GIS) is a computer system that analyzes and displays geographically referenced information. It uses data that is attached to a unique location.

What is ArcGIS Online?

ArcGIS Online is a cloud-based mapping and analysis solution. Use it to make maps, analyze data, and to share and collaborate.

UC Berkeley provides free ArcGIS Online.
