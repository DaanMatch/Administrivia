# Prerequisites
```
pip install -U jupyter-book
pip install ghp-import
```


## Updating Jupyter Book
Rebuild HTML files 
```
jb build <book name>
```
Push HTML content to GitHub pages branch
```
ghp-import -n -p -f _build/html
```
